    Mis Proyectos

    Simulación de red cliente-servidor y cableado estructurado

    En este primer proyecto, hemos formado un equipo de entre 3 y 6 personas y hemos realizado una red que simulaba la conexión entre un cliente a un servidor. Este cliente está conectado a un router que se sitúa en la delegación en Barcelona, que a la vez, se conecta al router de la delegación en Madrid, que es donde está conectado el servidor. También hemos montado un cableado estructurado (con canaletas y cables RJ45 CAT 6), para la red de ordenadores.

    A través de este proyecto, puedo demostrar habilidades en el trabajo en equipo, la capacidad de la organización y gestión de tareas, conocimientos en redes informáticas, cableado estructurado, sistemas operativos como Linux, además de la creación de una página web.

    Robot Roomba

    En este proyecto hemos montado un robot que se mueve y se comporta igual que un Roomba. Básicamente, este robot está hecho con un tupper, donde dentro tiene una batería, un motor que usa el robot para moverse con unas ruedas, luces led, un sensor que usa el robot como sus ojos y un montón de cables conectados a una placa Arduino.

    Lo que hace este robot es ir hacia delante, si se encuentra un obstáculo en medio, ya sea una pared o cualquier objeto, se detiene para encender sus luces indicándonos que ha detectado un obstáculo y se gira para ir hacia otro lado.

    Este proyecto nos permitió adentrarnos en el funcionamiento interno de los robots Roomba, y nos enseñó cómo programar uno con Arduino. Además, reflexionamos sobre las posibles aplicaciones prácticas de este tipo de tecnología en la vida diaria, como su uso en el hogar, en la asistencia a personas con necesidades especiales, o incluso para la automatización de tareas.

    Sistema informático para una empresa de Ciberseguridad

    Para este proyecto hemos creado un equipo de 3 personas donde cada uno ha tenido que trabajar para ser experto en tres áreas diferentes: Sistemas operativos, Redes informáticas y Robótica.

    En Sistemas Operativos hemos tenido que preparar unos servicios (DHCP, DNS y Active Directory) en sistemas operativos tanto Ubuntu Server y Windows Server.

    En Redes, hemos montado una red en el programa Cisco Packet Tracer que simulaba una red para una empresa de ciberseguridad, donde había montado unos cuantos servidores y ordenadores cliente.

    En Robótica hemos creado un “aparato” que podía detectar si la temperatura ambiente de la habitación donde estaban situados los servidores supera los 30º C o no. Este aparato disponía de un sensor de temperatura, luces led y un buzzer que emite sonidos.

    El objetivo de este proyecto era que cada uno se hiciera experto en una de las áreas que mencioné anteriormente para después enseñar al resto del equipo cómo funcionaba o cómo se hacía lo que hacía cada uno en esa área, de manera que todos podíamos juntar nuestros conocimientos.

    A través de este proyecto, puedo demostrar la capacidad de poder aprender y adquirir conocimiento por nuestra cuenta, donde demostramos la capacidad de enseñar y compartir nuestros conocimientos, lo que implica habilidades de comunicación efectiva y capacidad de trabajar en equipo de manera colaborativa para lograr un objetivo común.

    Página web para una empresa de ciberseguridad

    En este proyecto, he desarrollado una página web para una empresa ficticia de ciberseguridad, donde he creado una página de inicio, el inicio de sesión del usuario, el registro y un panel de administración para, obviamente, los administradores de la empresa.

    En este proyecto he utilizado varios lenguajes de programación, frameworks y otras tecnologías como por ejemplo, HTML y CSS para crear el diseño de la página, JavaScript para hacer la página más funcional e interactiva, el servicio MySQL para alojar una base de datos donde guardar los datos de usuarios, PHP para poder enlazar la página con la base de datos y entre otros.

    Este es un proyecto que me he propuesto por mi cuenta para poner en práctica mis habilidades y mis conocimientos sobre el desarrollo de páginas web, tanto en la parte de front-end (en la parte del usuario) como en la parte de back-end (en el lado del servidor).

    Con este proyecto consigo demostrar mis habilidades y capacidades para trabajar en proyectos de manera autónoma, poner en práctica habilidades y conocimientos previamente adquiridos y abordar problemas complejos.
